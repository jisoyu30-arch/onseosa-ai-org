칼의 피드백이 모호하고 반복적임 (매번 62점, 같은 피드백).

**Why:** 노아가 빈 데이터로 분석해서 근거가 없고, 칼이 레퍼런스 없이 주관적으로 평가하기 때문.

**How to apply:**
1. 노아에게 유튜브 레퍼런스 링크를 입력해서 실제 트렌드/조회수 패턴 분석
2. 노아 분석 결과를 칼에게도 전달 → 근거 기반 검수
3. 칼 모델을 GPT-4o → 상위 모델(o1 또는 GPT-4.5)로 변경하여 정밀 검수
4. 칼 피드백에 레퍼런스 예시 포함 → 리아가 구체적으로 수정 가능
