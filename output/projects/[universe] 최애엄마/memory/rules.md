# 절대 규칙 (Do Not Break)

## 설정 규칙
1. 은주는 33세 방송작가. 나이 변경 불가
2. 빙의체는 강은영. 개명 없음
3. 한별 회귀 시점 나이는 12살
4. 한별 데뷔 나이 16살 (중2 캐스팅 → 3년 연습)
5. 한별 사망 나이 25살 (회귀 전 타임라인)
6. 이섭 = 제이든 = 제피르 동일 인물. 분리 불가
7. 강은영은 입양아 (후반부 반전. 조기 노출 금지)
8. SNS 플랫폼 이름: 아웃문그램 (Outmoongram)
9. 물주 비번 1230 = 한인아 생일 12월 30일

## 톤 규칙
10. 무겁되 가볍게, 아프되 웃기게
11. 블랙코미디와 감정 드라마가 한 문장 안에 공존
12. 덕후의 언어로 쓰는 육아물 — 이론이 아니라 덕질 감각으로

## 구조 규칙

14. 크레톡스 팬덤명은 ANTIDOTE
15. 통장 47,320원에서 시작 — 치트키 없음
16. 미래 지식 + 콘텐츠 감각이 유일한 무기

## 관계 규칙

18. 우주는 은주(엄마)를 쉽게 받아들이지 않음

20. 우주가 류준서를 짝사랑 (250화 이후)
