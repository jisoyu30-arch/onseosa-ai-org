# 눈떠보니 최애 엄마?! — 캐릭터

## 주요 인물

### 김은주 (주인공 / 방송작가 / 33세)
- 방송작가 10년차. 일은 잘하지만 자기 삶은 삭막
- 어린 시절 엄마 부재 → 결핍, 비혼주의
- 9년간 아이돌 덕질로 빈자리 채움
- 한별 자살 → 죄책감 → 막걸리 → 회귀
- 빙의체: 강은영(39세, 한별 친모)
- 강점: 콘텐츠 감각, 아이돌 산업 이해, 미래 지식

### 강은영 (빙의체 / 한별 친모 / 39세)
- 입양아 (후반부 반전으로 밝혀짐)
- 원래: 미모 출중하나 인성 파탄, 자식 학대
- 남편 사고 후 시아주버니에게 사업체 뺏김 → 거액 빚 + 아이들
- 외로움 → 남자 문제 → 자식을 돈벌이 수단으로 착취
- 은주 빙의 후: 완전히 다른 사람으로 변화

### 한별 (최애 / 이카루스 리더 / 12세→25세)
- 순하고 다정하고 눈치 빠름
- 상처받는 데 익숙하지만 비뚤어지지 않음
- 작은 친절에 크게 흔들리는 아이
- 아역배우 → 연습생 → 이카루스 데뷔(16세)
- 회귀 전: 26세에 자살 (친구 마약사망 루머 → 검찰조사)
- 가장 큰 고통: 학대하는 친엄마의 존재

### 한우주 (한별 누나 / 중2 / 15세)
- 아빠 있던 시절을 기억 → 더 아픔
- 엄마를 전혀 믿지 않음, 분노와 냉소
- 엄마 닮은 자신이 싫어 자기파괴적
- 변한 엄마를 쉽게 받아들이지 않음
- 은주에게 한별보다 훨씬 어려운 상대

### 이섭 / 제이든 / 제피르 (동일 인물 / 12세 전후)
- 한별의 첫사랑이자 유일한 안식처
- 회귀 전: 마약 사망 → 한별 몰락의 시작점
- 이섭이 나쁜 게 아니라 구조가 망가뜨림
- 제이든 = 크레톡스 리더
- 제피르 = 블랙라이트 (약물 사망)

### 신 (회귀 설계자)
- 예닐곱 살 여자아이 형상
- 장난스럽고 얄밉고, 정답은 절대 안 가르쳐줌
- "죽지 않게 잘 키워요." 한마디가 전부
