# 전체 줄거리 구조

## 발단: 최애의 죽음과 회귀 (1~3화)
한별 26세 자살 → 은주 자책 → 막걸리 → 회귀 → 강은영으로 빙의 → 신 등장 "죽지 않게 잘 키워요"

## 전개1: 생존과 인플루언서 (4~30화)
통장 47,320원 → 물주 의존 끊기 → 아웃문그램 인플루언서 → 미래 지식으로 선점 → 우주와 기싸움 + 한별 눈치 + 생활고

## 전개2: 기획과 확장 (31~60화)
인플루언서 수입 안정 → 한인아 투자 → 기획사 설립 결심 → 매니저 이해수 영입 → 재능있는 아이들 캐스팅 → 류준서 등장 (혐관) → 39화 우주 위기 → 은주 구출 → 첫 화해

## 전개3: 최애 육성과 새 위기 (61~100화)
한별 14~15세 캐스팅 → 연습생 시작 → 한별이 이섭 데려옴 → 은주 격렬 반대 → 첫 충돌 → 결국 수용 → 70화 한별-이섭 비밀 목격 → 은주 기절

## 중반: 이카루스 데뷔와 업계 (101~250화)
한별 17세 이카루스 데뷔 → 한별-이섭 비밀 커플 → 업계 견제 → 류준서 대형기획사 이탈 → 은주 파트너 합류 → 250화 우주의 류준서 짝사랑 발각

## 후반: 균열과 구원 (251화~)
은주 몸 이상 → 280화 간암 말기 → 우주+한별 이식 희망 → 은주 거절 → 친정엄마 혈액검사 → 강은영 입양아 밝혀짐 → 은주 투병하며 계속 나아감

## 차별점
1. 덕후의 언어로 쓰는 육아물
2. 현실적 생존 서사 (통장 47,320원, 치트키 없음)
3. 아이돌 업계 내부 시선
4. 두 아이, 두 개의 다른 상처
5. 블랙코미디 + 감정 드라마 공존
